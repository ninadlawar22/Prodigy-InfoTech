HOW TO RUN:

1. Open MySQL Workbench.
2. Run database.sql file.
3. Open backend/.env file.
4. Replace DB_PASSWORD with your MySQL password.
5. Open terminal inside backend folder.
6. Run:
   npm install
   npm run dev

7. Open frontend/index.html in browser.
   Better: Use VS Code Live Server extension.

http://localhost:5000
