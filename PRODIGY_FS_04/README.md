# Real-Time Chat Application

A full stack real-time chat app using Node.js, Express, Socket.IO WebSocket technology, SQLite database, and a simple attractive frontend.

## Features

- User registration and login
- Password hashing using bcryptjs
- Real-time messaging using Socket.IO
- Join public chat rooms
- Private one-to-one chat
- Online user status
- Typing indicator
- Chat message history stored in SQLite database
- Simple, attractive, responsive UI

## Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- WebSocket: Socket.IO
- Database: SQLite
- Authentication: Express Session + bcryptjs

## Run Project

```bash
npm install
npm start
```

Open:

```text
http://localhost:3000
```

## Test Real-Time Chat

1. Open the app in one browser tab and create User 1.
2. Open the app in another browser or incognito window and create User 2.
3. Join the same room or select the user for private chat.
4. Send messages and see instant real-time communication.
