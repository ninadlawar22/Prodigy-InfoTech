# LocalStore E-Commerce Website

A full stack e-commerce website for a local store using Node.js, Express, EJS and SQLite database.

## Features
- Product listings with images, descriptions and prices
- Search, category filter and price sorting
- Shopping cart with update/remove quantity
- User registration and login authentication
- Checkout and order placement
- Order tracking page
- Customer reviews
- Customer support form

## How to Run
1. Install Node.js from https://nodejs.org/
2. Open this project folder in VS Code
3. Open terminal in VS Code
4. Run: npm install
5. Run: npm start
6. Open browser and visit: http://localhost:3000

The SQLite database is created automatically in the db folder when the project starts.
